# TCM Constitution Data Analysis
中医体质数据分析

## Project Overview
Used Python to analyze TCM constitution distribution and age patterns among 10 patients.

## Tools Used
- Python
- pandas
- matplotlib

## Key Findings
- Yang deficiency and Qi deficiency were the most common constitutions (3 patients each)
- Yang deficiency patients had the highest average age (45.3 years)
- Balanced constitution patients were the youngest (37.5 years average)

## Skills Demonstrated
Data cleaning, statistical analysis, data visualization
